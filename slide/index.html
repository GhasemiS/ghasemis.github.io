ERDA

ERDA (Electronic Research Data Archive) is AU’s digital solution for storing and sharing open research data.

  • Do not use ERDA if it contains:
    • Personal data
    • IP-data
    • Strategically important research data
  • Use SIF instead
  • Login using Aarhus University email and password

  • Snapshots are taken on ERDA’s primary site every 15 minutes. These are saved for 8 days.
  • ERDA is copied to a secondary site (>14 km distant from the primary site) every 24 hours.
  • Snapshots are taken on the secondary site once a day. These are saved for 30 days.
  • You can secure with 2FA.

Setting Up Password for SFTP & WebDAV

https://int.erda.au.dk/wsgi-bin/setup.py

Set a password that you will remember
(preferably not your AU password)

Install SSHFS-win

  1. Request Admin Privilege in Heimdal Agent
  2. Open PowerShell and type:

winget install SSHFS-Win.SSHFS-Win

Open Windows File Manager.
Right click on “This PC” and select Map network drive.

Enter the following in the Folder field:


\\sshfs\YOUREMAIL@biomed.au.dk@io.erda.au.dk!2222

Enter your password if required.

for WebDAV:

In Windows open Map Network Drive and in the folder section enter:


https://io.erda.au.dk:4443

Username: Biomed Email
Password: ERDA password

On MacOS search for Connect to Server. Then in the Server address field enter:


https://io.erda.au.dk:4443
